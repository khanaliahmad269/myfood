# MyFood

MyFood is a full-stack web application for browsing and ordering junk and fast food.

## Features

- Browse a variety of junk and fast food items.
- Place orders for selected food items.
- Contact form for customer inquiries.
- Responsive design for mobile and desktop.

## Tech Stack

- **Frontend**: React, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: SQLite (mocked with JSON files for this demo)

## Setup Instructions

### Frontend

1. Navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

### Backend

1. Navigate to the `backend` directory:
   ```bash
   cd backend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the server:
   ```bash
   npm start
   ```

### Environment Variables

Create a `.env` file in the `backend` directory and configure the following variables:

```
PORT=5000
```
